## 3. GOOGLE SEARCH CONSOLE — самое важное

Сайт сейчас **не виден в Google вообще**. Нужно подтвердить владение.

### Добавь этот мета-тег в `<head>` главной страницы:

```html
<meta name="google-site-verification" content="gaujHeXql8rMRmxFJG2u-dF0hFI1m3ZRNc_11Bxb26Y" />
```

Вставить в `<head>` до `<title>` или сразу после — не важно, главное внутри `<head>`.

### После деплоя:

1. Зайти на https://search.google.com/search-console
2. Добавить ресурс → "Ресурс с префиксом в URL" → `https://agro.remeta.com.br`
3. Выбрать способ "Тег HTML" → нажать "Подтвердить"
4. После подтверждения: Файлы Sitemap → добавить `sitemap.xml`
5. Проверка URL → отправить на индексацию главную и 4 статьи

## 7. ПОСЛЕ ДЕПЛОЯ — Bing IndexNow (необязательно, но полезно)

Запустить один раз из терминала:

```bash
curl -X POST "https://api.indexnow.org/indexnow" \
  -H "Content-Type: application/json" \
  -d '{
    "host": "agro.remeta.com.br",
    "key": "remeta2026agro",
    "urlList": [
      "https://agro.remeta.com.br/",
      "https://agro.remeta.com.br/guides/export-fishmeal-china",
      "https://agro.remeta.com.br/guides/gacc-mara-registration",
      "https://agro.remeta.com.br/fishmeal-buyers/india/guide",
      "https://agro.remeta.com.br/fishmeal-buyers/pakistan/guide"
    ]
  }'
```
